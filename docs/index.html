Quick Start

Chat Completions

POST https://cocapn.ai/v1/chat/completions

OpenAI-compatible chat completions endpoint. Drop-in replacement for any OpenAI SDK.

Request Body

FieldTypeDescription
model requiredstringModel ID (e.g., "deepseek-chat")
messages requiredarrayChat messages with role/content
stream optionalbooleanEnable streaming (default: false)
temperature optionalnumberSampling temperature (0-2, default: 1)
max_tokens optionalintegerMax tokens to generate
system optionalstringSystem prompt (alternative to system message)

Example

curl https://cocapn.ai/v1/chat/completions \
  -H "Authorization: Bearer cpn_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek-chat",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "What is the capital of France?"}
    ],
    "temperature": 0.7
  }'

Streaming

Set "stream": true to receive Server-Sent Events:

curl https://cocapn.ai/v1/chat/completions \
  -H "Authorization: Bearer cpn_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek-chat",
    "messages": [{"role": "user", "content": "Tell me a story"}],
    "stream": true
  }'

# Returns:
data: {"choices":[{"delta":{"content":"Once"},"index":0}]}
data: {"choices":[{"delta":{"content":" upon"},"index":0}]}
data: {"choices":[{"delta":{"content":" a time..."}],"index":0}]}
data: [DONE]

Bring Your Own Keys

Add your own provider API keys in the dashboard. Your keys are used first, server keys as fallback. Your keys never leave our servers (stored in Cloudflare Secrets Store).

🔑 Why BYOK?

Use your existing credits. No markup on your own keys. Full transparency on costs.

🛡️ Security

Keys stored in Cloudflare Secrets Store, never in Worker code or KV. Encrypted at rest.

Node.js SDK

npm install cocapn
import { chat, chatStream, models, usage } from 'cocapn';

// Basic chat
const response = await chat({
  apiKey: 'cpn_your_key',
  messages: [{ role: 'user', content: 'Hello!' }],
  model: 'deepseek-chat'
});
console.log(response.content);
console.log(response.costUsd);

// Streaming
for await (const chunk of chatStream({
  apiKey: 'cpn_your_key',
  messages: [{ role: 'user', content: 'Tell me a story' }]
})) {
  process.stdout.write(chunk.content);
}

// List models
const modelList = await models({ apiKey: 'cpn_your_key' });

// Usage stats
const stats = await usage({ apiKey: 'cpn_your_key' });

Python SDK

pip install cocapn
from cocapn import Client

client = Client(api_key="cpn_your_key")

# Basic chat
response = client.chat(
    messages=[{"role": "user", "content": "Hello!"}],
    model="deepseek-chat"
)
print(response.content)
print(response.cost_usd)

# Streaming
for chunk in client.chat_stream(
    messages=[{"role": "user", "content": "Tell me a story"}]
):
    print(chunk.content, end="", flush=True)

# List models
models = client.models()

# Usage
stats = client.usage()

Go SDK

go get cocapn-go
package main

import (
    "fmt"
    cocapn "cocapn-go"
)

func main() {
    client := cocapn.NewClient(
        cocapn.WithAPIKey("cpn_your_key"),
    )

    resp, err := client.Chat(
        cocapn.WithModel("deepseek-chat"),
        cocapn.WithMessages(
            cocapn.UserMessage("Hello!"),
        ),
    )
    if err != nil {
        panic(err)
    }
    fmt.Println(resp.Content)
    fmt.Printf("Cost: $%.6f\n", resp.CostUSD)
}
